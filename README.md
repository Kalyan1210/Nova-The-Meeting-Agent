# AI Meeting Agent

A self-hosted AI agent that joins video calls as an active participant — listens continuously, answers questions from your knowledge base, and executes actions during the call.

## Structure

- `docs/` — product thinking, meeting notes, brainstorming (Kurt: Obsidian | Kalyan: VS Code)
- `src/` — code

## Stack

Built on OpenClaw. Customers bring their own API keys (Claude, Whisper, ElevenLabs).
